Insertion Sort

An inplace sorting algorithm which is stable. Worst Case Complexity O(N2). Insertion sort is a simple sorting algorithm that builds the final sorted array one item at a time. It is much less efficient on large lists than more advanced algorithms such as quicksort, heapsort, or merge sort.
