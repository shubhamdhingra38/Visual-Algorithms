Graph Traversal

Breadth First Search and Depth First Search, two ways in which a graph can be traversed. BFS uses a queue and visits every neighbor first. DFS uses a stack and traverses the branch as deep as possible.

Dijkstra's SSP

A greedy algorithm to find the shortest path to every other vertex in a graph from a given source vertex.

Prim's MST

Prim's is a greedy algorithm that finds a minimum spanning tree for a weighted undirected graph.

Convolution2D

Convolution is the process of adding each element of the image to its local neighbors, weighted by the kernel. This is extensively used in image processing and ConvNets in Deep Learning.

Value Iteration

An algorithm in Reinforcement Learning, which finds the optimal action-value function. It can only be used when the MDP is fully known.

Genetic Algorithm

Inspired by Darwin's theory of evolution and survival of the fittest, a randomly generated population converges to an optima for some given task by reproduction.

Insertion Sort

A simple sorting algorithm which runs in O(n²). The technique followed is same as that a person uses when playing a game of cards.

Convex Hull

The gift wrapping algorithm produces the Convex Hull or the minimum set of points as the vertices of a polygon such that all other points lie inside.

Travelling Salesman

A classical NP-hard problem which consists of minimizing the overall distance visiting every city and coming back to initial city.

Linear Regression

A concept in statistics and machine learning to model the relationship between a dependent and independent variable.

Quick Sort

Quicksort is an efficient sorting algorithm. It has an average time complexity of O(nlogn). It uses divide and conquer.

A*

A* is a graph traversal and path search algorithm, which is often used in computer science due to its completeness, optimality, and optimal efficiency.
