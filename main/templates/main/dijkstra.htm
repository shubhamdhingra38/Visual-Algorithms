Dijkstra's algorithm is an algorithm for finding the shortest paths between nodes in a graph, which may represent, for example, road networks.

For a given source node in the graph, the algorithm finds the shortest path between that node and every other. It is a greedy algorithm.

It can be implemented in O(V²) by using Adjacency Matrix but this can be improved to O(|E|+|V|log |V|) by using a Fibonacci heap min-priority queue.
