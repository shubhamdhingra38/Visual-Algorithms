Breadth-first search (BFS) is an algorithm for traversing or searching tree or graph data structures. It starts at the tree root, and explores all of the neighbor nodes at the present depth prior to moving on to the nodes at the next depth level. It uses Queue data structure.

The opposite, Depth-first search (DFS), which instead explores the node branch as far as possible before being forced to backtrack and expand other nodes. It uses Stack data structure.
